# Dashboard-TN
Dashboard para o jornal Tribuna do Norte

## Requisitos

Python
```bash
  3.12
```

Libs
```bash
  pandas==2.1.3
  Pillow==10.1.0
  pygal==3.0.0
  Requests==2.31.0
  streamlit==1.28.2
```

Instale as bibliotecas
```bash
  pip install -r requirements.txt
```

Vá até a pasta /Dashboard-TN e execute
```bash
  streamlit run Dashboard.py
```
